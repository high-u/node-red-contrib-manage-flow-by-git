# node-red-contrib-manage-flow-by-git

## Released

|Date|Version|Description|
|:--:|:--:|:--|

## Feature

- Manage flow by git.
- Export XML.
- JSON for restoration is also included in XML.

## Install

- npm install --save node-red-contrib-manage-flow-by-git

## Usage

- Open "by git" tab.
  1. Select nodes.
  1. Get XML.
  1. Download XML.
